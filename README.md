# Regex Data Extraction - ALU Project

## 🔍 Overview
This project uses Python and Regular Expressions to extract structured data from large unstructured text. Patterns implemented include:

- Email addresses
- URLs
- Phone numbers
- Credit card numbers
- Currency amounts
