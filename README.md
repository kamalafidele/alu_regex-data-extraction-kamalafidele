# Data Extraction With Regex

## 🔍 Project Overview
> This project leverages Python and Regular Expressions to extract structured data from extensive unstructured text. The implemented patterns include:


- Phone numbers
- Credit card numbers
- Email addresses
- Currency amounts
- URLs
